# Documentation TaskFlow

Ce dossier regroupe la documentation du projet mobile **TaskFlow**.

## Documents disponibles

- `documentation-fonctionnelle.md` : besoins metier, fonctionnalites (dont authentification locale), parcours utilisateur, regles de gestion.
- `documentation-technique.md` : architecture Flutter, auth locale, API REST, tests, demarrage.
